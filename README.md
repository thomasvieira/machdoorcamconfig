
# Mach Door CAM Config

Addon/Plugin para Freecad para configuração dos parametros utilizados na geração de arquivos gcode para a Mach Door, produzida pela Mach CNC.


## Pré-requisitos

Freecad 1.0 devidamente instalado. Baixe gratuitamente de [freecad.org/downloads](https://www.freecad.org/downloads.php)

## Instalação

Esse plugin não requer um instalador. Apenas baixe e descompacte o arquivo [MachDoorCAMConfig.zip](https://github.com/thomasvieira/machdoorcamconfig/blob/main/MachDoorCAMConfig.zip) na pasta MOD do Freecad. Tipicamente no Windows o caminho para a pasta MOD é ***C:\Users\Username\AppData\Roaming\FreeCAD\Mod*** onde Username é o nome de usuário do Windows.
